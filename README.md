# 🤖 AI Content Automation System / Sistema de Automatización de Contenido con IA

An automated AI-powered content generation system built with Python.

Un sistema automatizado de generación de contenido con Inteligencia Artificial desarrollado en Python.

This project automatically generates:
- Topics
- Scripts
- Background music
- Videos
- Thumbnails
- Social media posts

Este proyecto genera automáticamente:
- Ideas de contenido
- Guiones
- Música de fondo
- Videos
- Miniaturas
- Publicaciones para redes sociales

---

# 🚀 Features / Características

### 🧠 AI Topic Generator
Automatically generates trending topics for content creation.

Genera automáticamente ideas de contenido basadas en tendencias.

---

### ✍️ Script Generator
Creates scripts for videos and social media posts using AI.

Genera guiones automáticamente para videos y contenido digital.

---

### 🌍 Translator
Translates content into different languages.

Permite traducir el contenido generado a diferentes idiomas.

---

### 🎵 Music Generator
Generates background music for videos.

Genera música de fondo automáticamente para videos.

---

### 🎬 Video Generator
Automatically creates videos using:

- Generated scripts
- Background music
- Visual assets

Crea videos automáticamente usando:

- Guiones generados
- Música de fondo
- Recursos visuales

---

### 🖼 Thumbnail Generator
Creates thumbnails optimized for social media.

Genera miniaturas optimizadas para redes sociales.

---

### 📲 Social Media Post Generator
Generates captions and posts for platforms such as:

- Instagram
- LinkedIn
- Short video platforms

Genera publicaciones automáticas para plataformas como:

- Instagram
- LinkedIn
- Plataformas de videos cortos

---

### ☁️ Cloud Upload
Uploads generated videos to Cloudinary.

Sube automáticamente los videos generados a Cloudinary.
